## Project Overview

This project focuses on the design and development of a professional website for Amani Gaming Hub. The business is a local gaming retail and entertainment hub that sells consoles, accessories, and games, while also offering in-store gaming services. The purpose of the website is to improve the business’s online presence, provide clear information about products and services, and create a better experience for customers. The current website is limited, and this project aims to redesign it with improved structure, modern design, and better functionality.

## Website Goals and Objectives

The main goal of the website is to improve the online presence of Amani Gaming Hub and support business growth. The website will help attract new customers, display products clearly, and improve communication with customers.

Objectives include:

Increase website traffic
Improve product visibility
Increase customer enquiries
Support in-store visits
Improve overall user experience
